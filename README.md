# Caterpillar

A Boilerplate Node.js Express REST API with Typescript

## Installation

Clone the repo
```bash
git clone https://github.com/BrunoWilkinson/Catterpilar.git
```

Install
```bash
yarn 
```

To start working :D
```bash
yarn watch-ts
docker-compose up
```
Listen to file change, no need to do any else ;)
